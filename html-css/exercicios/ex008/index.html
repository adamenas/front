Texto marcado

Texto pode ser marcado com a tag <mark> e complementando com o style, que já é CSS. Essa alteração, porém, é local. Outra marcação não herdará a cor.
Para herdar a cor deverá ser definido o style no head, direto para o parâmetro mark, de modo a universalizá-lo
